Методы

конструктор промиса результат передается аргументом в .then ошибка передается аргументом в .catch
выполняется обработчик onFulfilled result -> функция или код при успешном выполнении
выполняется обработчик onRejected error -> функция или код при неуспешном выполнении
возвращает промис, который ждёт, пока все переданные промисы завершатся массив промисов новый промис, который резолвится, когда резолвятся все промисы из списка
результатом будет только первый успешно выполнившийся промис из списка массив промисов первый промис из списка
создаёт успешно выполнившийся промис с результатом value value -> результат
создаёт уже выполнившийся промис с ошибкой error value -> ошибка
