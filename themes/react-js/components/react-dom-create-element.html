React.createElement

  • Код, написанный с использованием JSX, будет преобразован в React.createElement()
  • React.createElement - создаёт и возвращает новый React-элемент определённого типа
    • type - строка, содержащая имя тега (например, 'div' или 'span'), React-компонент (класс или функция) или React-фрагмент
    • [props]
    • [...children]
