Списки и ключи

  • У каждого элемента массива должен быть ключ (key). «Ключ» - это специальный строковый атрибут, который нужно указывать при создании списка элементов. Ключи помогают React определять, какие элементы были изменены, добавлены или удалены. Их необходимо указывать, чтобы React мог сопоставлять элементы массива с течением времени. Если вы опустите ключ для элемента в списке, то React по умолчанию будет использовать индексы как ключи
  • Ключи внутри массива должны быть уникальными только среди своих соседних элементов. Им не нужно быть уникальными глобально. Можно использовать один и тот же ключ в двух разных массивах
  • Когда у дочерних элементов есть ключи, React использует их, чтобы сопоставить потомков исходного дерева с потомками последующего дерева

const arr = [1, 2, 3, 4, 5];
const items = arr.map((item) =>
	<li key={item.id}>
		{item.text}
	</li>
);
