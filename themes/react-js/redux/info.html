Flux

    Методология Flux состоит из 4 частей
  1. View (компоненты)
  2. Store (хранилище)
  3. Dispatcher - сообщает Store о каком-то событии и передает информацию
  4. Action (события)

Redux

  • Функция Reducer (преобразователь) - старое значение состояния преобразуется в новое
