shouldComponentUpdate()

  • shouldComponentUpdate(nextProps, nextState) для указания необходимости следующего рендера на основе изменений состояния и пропсов. По умолчанию происходит повторный рендер при любом изменении состояния. Вызывается перед рендером, когда получает новые пропсы или состояние. Необходим только для повышения производительности
