Конструктор new

  • new недоступно у стрелочной функции

Функция, запущенная через new

  1. Создаётся новый пустой объект через Object.create
  2. this получает ссылку на этот объект
  3. Модифицируется this (т.е. этот новый объект) и инициализируются методы и свойства
  4. Создается свойство __proto__ в которое записывается ссылка из prototype функции-конструктора (устанавливается прототип)
  5. Неявно возвращается this без вызова return
