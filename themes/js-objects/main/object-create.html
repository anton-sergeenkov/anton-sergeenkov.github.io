Object.create и Object.assign

Object.create

  • Object.create добавление прототипа при создании объекта (создаёт пустой объект с __proto__, равным первому аргументу, второй содержит дескрипторы свойств)
    • proto - __proto__
    • descriptors - дескрипторы свойств

Object.assign

  • Object.assign используется для копирования значений всех собственных перечисляемых свойств (при обходе в цикле for..in его можно будет получить) из одного или более исходных объектов в целевой объект. После копирования он возвращает целевой объект
    • obj - исходный объект

let obj = Object.assign(
	{ name: 'Tony' }, 
	{ age: 25 }
);
// obj {name: "Tony", age: 25}
