Прототип __proto__ [[Prototype]]

  • __proto__ - специальная ссылка (у объекта) на другой объект (прототип)
  • __proto__ - ссылка на объект функции конструктора свойства prototype
  • Если при чтении свойства, оно отсутствует в самом объекте, оно ищется в объекте __proto__
  • __proto__ - наследование на уровне объектов

function User() {
    this.name = 'Tony';
}
User.prototype.myFunc = function() {}
Object.prototype.myFuncObject = function() {}
let obj = new User();

js
1. У каждой функции по умолчанию есть свойство prototype, содержащее объект
function User() {
    this.name = 'Tony';
}

2. Создание объекта через new
let user = new User();

3. В user {__proto__: ...} записывается ссылка из prototype функции-конструктора (User)
