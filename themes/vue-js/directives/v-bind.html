v-bind:

  • Связывание атрибутов элементов
  • Указание дополнительных значений
  • CSS-классы
  • CSS Style

Связывание атрибутов элементов

  • Директива связывает входной атрибут тега или входной параметр компонента с выражением
  • Значение атрибута берется из переменной

<a v-bind:href="url">{{ link }}</a>  # полная запись
<p v-bind:class="activeClass"></p>   # полная запись
<p :class="activeClass"></p>         # сокращенная запись

Указание дополнительных значений


<div class="img" :style="{ backgroundImage: 'url('+item.img+')' }"></div>
<a :href="'courses/'+i"></a>

CSS-классы


<!-- active - название класса -->
<p :class="{ active:true }"></p>      # true (отобразится)
<p :class="{ active:false }"></p>     # false (не отобразится)
<p :class="{ active:isActive }"></p>  # свойство isActive (true/false)

Использование с обычным атрибутом class

  • Наличие класса active будет определяться истинностью параметра isActive
html
<div 
	class="static" 
	v-bind:class="{ active: isActive, 'text-danger': hasError }"
></div>

js
data: {
	isActive: true,
	hasError: false
}

result
<div class="static active"></div>

Массив в аргументе

html
<div :class="[activeClass, errorClass]"></div>

js
data: {
    activeClass: 'class-active',
    errorClass: 'class-error'
}

result
<div class="active text-danger"></div>

Объект в аргументе


<div v-bind:class="classObject"></div>


data: {
	classObject: {
		active: true,
		'text-danger': false
	}
}

Вычисляемое свойство в аргументе


<div v-bind:class="classObject"></div>


data: {
	isActive: true,
	error: null
},
computed: {
	classObject: function () {
		return {
			active: this.isActive && !this.error,
			'text-danger': this.error && this.error.type === 'fatal'
		}
	}
}

CSS Style

Объектная запись в HTML


<div :style="{ color: activeColor, fontSize: fontSize+'px' }"></div>


data: {
    activeColor: 'blue',
    fontSize: 30
}

Объектная запись в JS


<div :style="styleObject"></div>


// styleObject может содержать массивы или объекты
data: {
	styleObject: {
		width: '100px',
		height: '100px',
		background: 'blue'
	}
}

Использование массивов


<!-- Позволяет применить несколько объектов стилей к одному и тому же элементу -->
<div v-bind:style="[baseStyles, overridingStyles]"></div>
