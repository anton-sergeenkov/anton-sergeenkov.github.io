Отрисовка списков: v-for

  • Отображение массива элементов
  • v-for на компонентах
  • Когда присутствуют вместе на одном элементе, v-for имеет больший приоритет, чем v-if. Поэтому v-if будет выполняться для каждой итерации цикла
  • key с v-for всегда обязателен для компонентов

Отображение массива элементов

html
<li v-for="item in items">{{ item.name }}</li>
<li v-for="item of items">{{ item.name }}</li>
<li v-for="(item, i) in items">{{i}}, {{ item.name }}</li>
<li v-for="item in items" :key="item.id">{{ item.name }}</li>

js
items: [
	{name: 'Tony',  age: 25},
	{name: 'Tyler', age: 27},
	{name: 'Tom',   age: 20}
]


<div id="app">
	<ul>
		<!-- Без индексов -->
		<li v-for="car in cars">{{ car.model }}, {{ car.speed }}</li>
	</ul>
	<ul>
		<!-- С индексами -->
		<li v-for="(car, i) in cars">{{i}}, {{ car.model }}, {{ car.speed }}</li>
	</ul>
</div>


<div id="app">
    <ul>
    	<!-- Итерирование свойств объекта -->
        <li v-for="value in user">{{ value }}</li>
    </ul>
    <ul>
    	<!-- Использование второго аргумента для получения ключей объекта -->
        <li v-for="(value, key) in user">{{ key }}: {{ value }}</li>
    </ul>
    <ul>
    	<!-- Использование третьего аргумента для получения индексов объекта -->
        <li v-for="(value, key, index) in user">{{ index }}. {{ key }}: {{ value }}</li>
    </ul>
</div>


var app = new Vue({
	el: '#app', 
	data: {
		show: false,
		cars: [
			{model: 'Bmw',    speed: 250},
			{model: 'Audi',   speed: 350},
			{model: 'Nissan', speed: 270},
			{model: 'Ford',   speed: 150}
		],
		user: {
            name: 'Tony',
            age: 25
        }
	}
});

v-for на компонентах

  • При использовании v-for на компонентах обязательно требуется указывать key

<my-component v-for="item in items" :key="item.id"></my-component>
  • В компонент никакие данные не передаются автоматически, поскольку у компонентов изолированные области видимости. Для передачи итерируемых данных в компонент необходимо явно использовать входные параметры

<my-component
	v-for="(item, index) in items"
	v-bind:item="item"
	v-bind:index="index"
	v-bind:key="item.id"
></my-component>
