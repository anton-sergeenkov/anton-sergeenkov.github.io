CSS Grid Layout

Технологии разметки


1. CSS Flexbox Box Layout
2. CSS Grid Layout
3. CSS Box Alignment

Определения


- Grid Track           - расстояние между ближайшими двумя линиями, колонка или строка
- Grid Line            - линия, создаваемая Grid Track. grid-gap толщина линии
- Grid Cell            - ячейка сетки
- Grid Area            - область всегда прямоугольная. объединение нескольких ячеек в одну
- Line-based placement - позиционирование на основе линий

Сетка (родитель)


div {
	display: grid; - объявление	
}



div {
	grid-template-columns: 300px 300px 300px;             - столбцы (3 штуки)
	grid-template-rows: 150px 150px 150px;                - строки (3 штуки), можно не задавать. адаптируются под высоту контента
	grid-template: 150px 150px 150px / 300px 300px 300px; - короткая запись строки / столбцы
	grid-auto-columns: 200px;                             - размер по умолчанию для неявно заданных столбцов
	grid-auto-rows: 200px;                                - размер по умолчанию для неявно заданных строк
}



div {
	grid-gap: 20px;         - расстояние между строками и стобцами (толщина линии)
	grid-column-gap: 20px;  - расстояние между столбцами
	grid-row-gap: 20px;     - расстояние между строками	
}



div {
	grid-auto-flow: row;    - порядок элементов по очереди по строкам (по умолчанию)
	grid-auto-flow: column; - порядок элементов по очереди по столбцам
}
Для grid-template-columns и grid-template-rows 

300px 300px 300px;                     - 3 элемента (фиксированные величины)
1fr 1fr 1fr;                           - 3 элемента (относительные величины) fr - фракция
500px 1fr 2fr;                         - 3 элемента (смешанные величины)
500px auto 500px;                      - 3 элемента (auto - занимает ширину контента)
repeat(3, 1fr);                        - задать повторяющийся паттерн
minmax(150px, 250px) 1fr 1fr;          - 3 элемента, первый с minmax
minmax(100px, auto);                   - определяется автоматически по ширине элементов
repeat(auto-fill, 200px);              - добавлять столько элементов фиксированых размеров, сколько поместится в контейнер
repeat(auto-fill, minmax(150px, 1fr)); - когда минимальное значение 150px перескакивают колонки (может остаться пустое пространство сбоку)
repeat(auto-fit, minmax(150px, 1fr));  - когда минимальное значение 150px перескакивают колонки (пустое пространство не оставляет, блок заполняет всю ширину)



repeat()    - задание повторяющегося паттерна
minmax()    - задание минимального и максимального значения
auto-fill() - растяжение дочерних grid элементов на всю ширину родителя, если хватает элементов
auto-fit()  - растяжение дочерних grid элементов на всю ширину родителя
fr можно применять к высоте, только если у родителя задана высота

Объединение ячеек

Обращение к линиям по номеру

Задано 

.cards {
	display: grid;
	grid-template-columns: 1fr 1fr 1fr; // 3 столбца (4 линии)
	grid-template-rows: 200px 200px;    // 2 строки (3 линии)         
}

Позиционирование первого дочернего элемента (объединение 2х столбцов и 2х строк)

Объединяем столбцы от 2 линии до 4 линии

Объединяем строки от 1 линии до 3 линии


// полная запись
.card:nth-child(1) {
	grid-column-start: 2;
	grid-column-end: 4; // если написать -1 будет на всю ширину
	grid-row-start: 1;
	grid-row-end: 3;
}



// короткая запись
.card:nth-child(1) {
	grid-column: 2 / 4;
	grid-row: 1 / 3;
}



// сокращенная запись
.card:nth-child(1) {
	grid-area: 1 / 2 / 3 / 4;
}


grid-area:
- grid-row-start
- grid-column-start
- grid-row-end
- grid-column-end
Если элемент занимает 1 столбец или строку, то линию окончания указывать не нужно 

grid-column: 2 / 4;
grid-row: 1;
span - количество ячеек, которые нужно захватить 

grid-column: 1 / span 2; от 1 до 3 линии (занять 2 ячейки)
// аналогично
grid-column: 1 / 3;      от 1 до 3 линии

Обращение к линиям по имени

(результат аналогичен) 

.cards {
	display: grid;
	grid-gap: 20px;
	grid-template-columns: [side-start] 1fr
                           [main-start] 1fr 1fr
                           [main-end];
	grid-template-rows:    [main-start] 200px 200px
                           [main-end];                         
}
.card:nth-child(1) {
 	grid-column: main-start / main-end;
 	grid-row: main-start / main-end;
}

Обращение к линиям по имени (создание Grid Area)

(результат аналогичен)

Имена выбираются провизвольно, но если приписывать к именам -start или -end будет создаваться Grid Area.

В .cards указали main-start и main-end для строк и столбцов, поэтому grid-area: main;


.cards {
	display: grid;
	grid-gap: 20px;
	grid-template-columns: [side-start] 1fr
                           [main-start] 1fr 1fr
                           [main-end];
	grid-template-rows:    [main-start] 200px 200px
                           [main-end];                         
}
.card:nth-child(1) {
	grid-area: main;
}

Обращение по Grid Area


.cards {
	display: grid;
	grid-gap: 20px;
	grid-template-columns: repeat(3, 1fr);
	grid-template-rows: 200px 200px;  
	grid-template-areas: "header  header  menu"
						 "content content content";     
}
.card:nth-child(1) {
	grid-area: header;
}
.card:nth-child(2) {
	grid-area: menu;
}
.card:nth-child(3) {
	grid-area: content;
}
Если нужно оставить область пустой, ставим "." 

div {
	grid-template-areas: "header header  menu"
					     "  .    content content";
}

Использование именованных линий, не заданных явно


.wrapper {
	grid-column-gap: 2em;
	grid-row-gap: 20px;
	grid-template-columns: 2fr 4fr;
	grid-template-areas: 
		"header  header"
		"sidebar content"
		"footer  footer";
}
// мы определили область content и автоматически появились линии content-start и content-end
.test {

	background-color: blue;
	grid-column: content-start;
	grid-row: content-start / footer-end;
}

Выравнивание элементов

У родительского элемента (горизонтальное) 

div {
	justify-items: stretch; - блоки растягиваются по ширине (по умолчанию)
	justify-items: start;   - по левому краю.  блоки не растягиваются (только по ширине контента) 
	justify-items: end;     - по правому краю. блоки не растягиваются (только по ширине контента)
	justify-items: center;  - по центру.       блоки не растягиваются (только по ширине контента)
}
У родительского элемента (вертикальное) 

div {
	align-items: stretch;   - блоки растягиваются по высоте (по умолчанию)
	align-items: start;     - сверху
	align-items: end;       - снизу
	align-items: center;    - по центру
}
У дочернего элемента (горизонтальное) 

div {
	justify-self: stretch;
	justify-self: start;
	justify-self: end;
	justify-self: center;
}
У дочернего элемента (вертикальное) 

div {
	align-self: stretch;
	align-self: start;
	align-self: end;     
	align-self: center;	
}
