Операции над цветами


lighten($color,10%)         - осветляет исходный цвет
darken($color,10%)          - затемняет исходный цвет
saturate($color,70%)        - увеличивает насыщенность цвета от исходного
desaturate($color,70%)      - уменьшает насыщенность цвета от исходного
transparentize($color,0.2)  - добавляет к исходному цвету альфа-канал
opacify($color, 0.1)        - задание менее прозрачного цвета
transparentize($color, 0.4) - задание более прозрачного цвета
rgba(#333, 0.75)            - перевод из hex в rgba

Миксины, примеси (@mixin)

Позволяют сделать блок стилей, который может быть использован повторно в любом месте документа 

@mixin large-text {
	font:{
		family: Arial;
		size: 30px;
		weight: bold;
	}
	color: red;
}
p {
	@include large-text;
}



p {
	font-family: Arial;
	font-size: 30px;
	font-weight: bold;
	color: red;
}
Миксины можно подключать друг в друга 

@mixin large-text {
	...
	@include small-text;
}
Миксины могут принимать аргументы, которые передаются при подключении миксина 

@mixin my-border($color, $width) {
	border:{
		color:$color;
		width:$width;
		style:dashed;
	}
}
p {
	@include my-border(red, 2px);
}



p {
	border-color: red;
	border-width: 2px;
	border-style: dashed; 
}
Значение аргумента по умолчанию 

@mixin my-border($color, $width:10px) {
	border:{
		color:$color;
		width:$width;
		style:dashed;
	}
}
p {
	@include my-border(red);
}

Импорт других SASS файлов (@import)

Сконвертируется в css как @import если


1. Расширение css
2. Media-параметры
3. Присутствует http
4. Присутствует url



@import "main.css";
@import "main" screen;
@import "http://foo.com/main";
@import url(main);



@import url(main.css);
@import "main" screen;
@import "http://foo.com/main";
@import url(main);

Импортируется в sass файл если


1. Расширение sass                           @import "main.sass";
2. Расширение scss                           @import "main.scss";
3. Файлы css (main.css)                      @import "main";
4. Файлы-фрагмены (_main.sass / _main.scss)  @import "main";



@import "example.scss";
@import "example1", "example2";



/* файл example.scss */
.example {
	color: green;
}
/* файл style.scss */
#main {
	@import "example";
}



/* файл style.css */
#main .example {
	color: green; 
}

Директива @each

Если требуемся пройтись по списку значений

Устанавливает переменную в каждое из значений списка или карты и выводит содержащиеся в ней стили


@each $animal in cat,dog,mouse,tiger,wolf {
	.#{$animal}-icon {
		background-image:url(/img/#{$animal}.png);
	}
}



.cat-icon   { background-image: url(/img/cat.png); }
.dog-icon   { background-image: url(/img/dog.png); }
.mouse-icon { background-image: url(/img/mouse.png); }
.tiger-icon { background-image: url(/img/tiger.png); }
.wolf-icon  { background-image: url(/img/wolf.png); }

Директива @at-root

Изымает одно или несколько правил из родительского селектора в корневой уровень документа. Можно использовать с селекторами первого уровня вложенности.

Поднимает содержимые в ней правила в корень, отменяя цепочку от родителя


.parent {
	color: red;
		@at-root .child{
			color: green;
		}
}



.parent {
	color: red; 
}
.child {
	color: green; 
}



.parent {
	color: red;
		@at-root {
			.child1{
				color: green;
			}
			.child2{
				color: orange;
			}
		}
}



.parent {
	color: red; 
}
.child1 {
	color: green; 
}
.child2 {
	color: orange; 
}

Циклы For, While


@for $i from through - диапазон включает значение начала и конца
@for $i from to      - диапазон включает значение начала и не включает значение конца



@for $i from 1 through 3 {
	.item-#{$i} {
		width: 2em * $i;
		background-image: url(img/#{$i}.jpg);
	}
}









.item-1 {
	width: 2em; 
	background-image: url(img/1.jpg);
}
.item-2 {
	width: 4em;
	background-image: url(img/2.jpg);
}
.item-3 {
	width: 6em;
	background-image: url(img/3.jpg);
}



$i: 7;
@while $i > 0 {
	.item-#{$i} {
		width: 1em * $i;
	}
	$i: $i - 3;
}





.item-7 {
	width: 7em;
}
.item-4 {
	width: 4em;
}
.item-1 {
	width: 1em;
}
