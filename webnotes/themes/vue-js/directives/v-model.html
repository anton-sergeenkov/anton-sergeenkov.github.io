v-model

Двунаправленное связывание данных с элементами

(н-р input и textarea) 

<input type="text" v-model="messText">
<p>{{ messText }}</p>

<input type="checkbox" v-model="view">
<label>{{ view }}</label>
Аналогично 

<input type="text" @input="messText = $event.target.value">
<p>{{ messText }}</p>

Модификаторы


<!-- Удаление пробелов -->
<input type="text" v-model.trim="messText">
