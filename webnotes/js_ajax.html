JavaScript (Ajax jQuery) -> PHP (Метод GET, POST)

Короткая запись 

$.get("scripts/script.php", {name:name}, function(data) {
	// 
});



$.get("scripts/script.php", {name: "Tom", time: "2pm"}, function(data) {
	// 
});
Полная запись 

$.ajax({
	type: "GET",
	url: "scripts/script.php",
	data: { name:name, message:message }, 
	async: true,  
	success: function (data, textStatus) {
		console.log(data);
	}          
});

Для передачи может применять как метод GET (для небольших или средних записей), так и метод POST (для изображений или очень больших записей)

PHP (разбор данных)


$name    = $_GET[name]; 
$message = $_GET[message];

В зависимости от того, каким методом передавали (GET или POST), читать нужно из соответсвующего массива ($_GET, $_POST)
