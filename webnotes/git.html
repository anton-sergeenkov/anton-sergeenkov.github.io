Базовые операции


$ git clone                               # создание локальной копии удаленного git-репозитория (выполняется 1 раз)
$ git add .                               # добавление всех измененных файлов в коммит
$ git commit -m "comment"                 # сохранение изменений в локальном репозитории
$ git push                                # отправка изменений на удаленный git-репозиторий
$ git pull                                # получить изменения из удаленного git-репозитория в локальный репозиторий


$ git checkout -b features                # создание новой ветки "features"
$ git branch                              # текущая ветка
$ git checkout master                     # переключение на ветку "master"
$ git push --set-upstream origin features # 
$ git branch --delete features            # удаление ветки


$ git status                              # информация о текущей ветке, подготовленных коммитах и т.д.
$ git init                                # 
$ git checkout                            # 
$ git reset                               #

Git


1. Install Git
2. Open Git Bash
	$ cd путь_до_папки ($ cd и перетащить папку)

2. Generating a new SSH key


Generating a new SSH key


1. Generating a new SSH key
	$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
2. Adding your SSH key to the ssh-agent
	$ eval $(ssh-agent -s)
3. Add your SSH private key to the ssh-agent
	$ ssh-add ~/.ssh/id_rsa

3. Add the SSH key to your GitHub account


Adding a new SSH key to your GitHub account


1. Copy the SSH key to your clipboard.
	$ clip < ~/.ssh/id_rsa.pub
2. https://github.com/ -> Settings -> SSH and GPG keys -> New SSH key
	- ввести название ключа
	- вставить ключ скопированный в буфер обмена на 1 этапе
3. https://github.com/ -> Репозиторий -> Clone or download -> Use SSH -> Скопировать строку
4. Git Bash
	$ git clone git@github.com:name_account/name_repo.git

SmartGit

Откат изменений кода


delete  - удаляет файл из локального репозитория, если не было commit
discard - отказаться от последнего изменения файла, если не было commit

log -> ветка commit -> пкм -> revert -> Revert & Commit - откатить существующий commit
log -> ветка commit -> пкм -> reset (у последнего запушенного файла) -> set the index... - откат всех последних изменений и загрузка из удаленного git-репозиторя

Работа с ветками


Branches -> Local Branches -> пкм -> Add Branch -> Feature -> Add Branch
Branches -> Feature -> двойной клик переход на ветку

Branches -> Local Branches -> Feature (Выделить ветку в которую делаем слияние) -> двойной клик
Branches -> Local Branches -> Master -> пкм -> Merge -> Create Merge-Commit - слияние веток Feature и Master

Branches -> Local Branches -> пкм по ветке -> Delete - удалить ветку
