Анимация

hide(); show();


Скрывает/показывает элементы на странице (за счет плавного изменения его размера и прозрачности)
$('#box').hide(1000); // скрыть элемент 
$('#box').show(1000); // показать элемент
//1 - скорость исчезновения (миллисекунды 1с = 1000мс)
//2 - функция, которая будет выполнена после завершения работы

$('#box').hide(1000).show(1000); // цепные функции

slideUp(); slideDown();


Разворачивает/сворачивает элементы на странице (за счет плавного изменения высоты элементов)

$('#box').slideDown(1000); // появление (элемент выезжает вниз)

slideToggle();

Объект события eventObject()


eventObject(); передает параметр с информацией
	screenX  - координаты щелчка по оси X (от края монитора)
	screenY  - координаты щелчка по оси Y (по высоте)
	pageX    - координаты щелчка по оси X (для браузера)
	pageY    - координаты щелчка по оси Y (для браузера)
	altKey   - была ли нажата клавиша ALT
	ctrlKey  - была ли нажата клавиша CTRL
	shiftKey - была ли нажата клавиша SHIFT
	target   - какой объект отреагировал на событие



$('#box').click(function(eventObject) {
    alert(eventObject.shiftKey);
});



$('#box').click(function(eventObject) {
    var myObject = eventObject.target; // сохранили объект
    alert(myObject.id); // вывести значение атрибута ID для объекта
});
